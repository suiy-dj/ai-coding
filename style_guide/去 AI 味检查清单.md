# 去 AI 味检查清单

## 第一轮：事实审校

- [ ] 事实准确（数据、时间、品牌名）
- [ ] 逻辑通顺（无前后矛盾）
- [ ] 结构完整（有开头、主体、结尾）
- [ ] 无冗余内容

## 第二轮：去 AI 味

### 删除套话
- [ ] 删除"首先、其次、总之"等连接词
- [ ] 删除"在当今...的时代"等套话
- [ ] 删除"值得注意的是"等废话

### 替换模糊词
- [ ] "非常重要" → 具体说明多重要
- [ ] "越来越多" → 给出具体数据
- [ ] "可能、也许" → 明确态度

### 口语化改造
- [ ] "由此可见" → "所以"
- [ ] "并非" → "不是"
- [ ] "此外" → "还有"

### 增加具体细节
- [ ] 加入具体场景/对话
- [ ] 加入具体数字
- [ ] 加入具体人名/品牌名

### 明确态度
- [ ] 没有"可能、也许、大概"
- [ ] 有明确的观点立场
- [ ] 不模棱两可

## 第三轮：细节打磨

### 段落检查
- [ ] 段落长度 15-25 字为佳
- [ ] 没有超过 30 字的长段落
- [ ] 每段 3-5 句

### 句子检查
- [ ] 长短句交替
- [ ] 30 字以上长句不超过 3 个
- [ ] 没有病句

### 节奏检查
- [ ] 读起来顺口
- [ ] 没有拗口的地方
- [ ] 可以适当断句

---

## 常见 AI 表达及修改

| AI 表达 | 修改为 |
|--------|--------|
| 在当今 AI 高速发展的时代 | AI 发展太快了 |
| 值得注意的是 | 说白了 |
| 综上所述 | 所以 |
| 具有重要意义 | 很有用 |
| 呈现出...的趋势 | 越来越... |
| 的可能性较大 | 应该会 |
| 不容忽视 | 得重视 |
| 应运而生 | 出来了 |

---

## 检查完成后

- [ ] 全文通读一遍
- [ ] 大声朗读检查语感
- [ ] 确认没有机械感
